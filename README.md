# Address Book Application (C)

A menu-driven Address Book application developed in C using file handling.
This project allows users to store, search, edit, and delete contact details.

## Features
- Add new contacts
- Search contacts by name or number
- Edit existing contact details
- Delete contacts
- List Contacts
- Persistent storage using files

## Technologies Used
- C Programming
- File Handling
- Linux (GCC) or WSL
